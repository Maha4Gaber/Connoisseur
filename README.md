# connoisseur2

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### install json server to run database

```
npm install -g json-server
```

### enter DB folder

```
 cd src\DB
```

### Run json server 

```
json-server --watch db.json
```
